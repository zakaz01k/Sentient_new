How to play

  1. The website will generate a random word that the player needs to guess.
  2. The player can input their guess into a form provided on the website.
  3. After submitting a guess, the website will display how close the guess is to the correct word. This can be done by indicating the number of matching letters or providing a similarity score.
  4. The player's objective is to guess the word correctly in the fewest number of attempts and as quickly as possible.
